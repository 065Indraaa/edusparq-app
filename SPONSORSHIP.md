## 🤖 Multi-Agent Pipeline — Bukan Sekadar Prompt

Saat mahasiswa minta "Buat bab 3 skripsi", EduSparq tidak langsung generate. Ini yang terjadi:

```
1. CLASSIFIER    → "Ini tugas complex, butuh pipeline lengkap"
2. CLARIFIER     → "Blockchain bagian apa? Konsensus atau smart contract?"
3. SPECIFIER     → Susun spec teknis: format, panjang, batasan, kriteria sukses
4. PLANNER       → Outline: 3.1 Arsitektur, 3.2 Algoritma, 3.3 Implementasi
5. TASKER        → Pecah jadi 5 atomic tasks
6. IMPLEMENTER   → Cari materi user (RAG) + cari jurnal Crossref + cek KBBI
                   → Generate BAB 3 dengan 5 referensi DOI real
7. REVIEWER      → Audit kualitas, score 0-100, revisi bila perlu
```

**Hasil:** Output yang terstruktur, berbasis referensi nyata, bukan karangan.
